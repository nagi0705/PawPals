# PawPals

## サイト概要

### サイトテーマ
ペットに関する情報や写真を共有できる場を提供するコミュニティサイト

### テーマを選んだ理由
私自身がペットを飼っており、ペットについて他のオーナーと交流したり、情報交換したいと思うことが多くありました。また、ペットに関するトラブルや悩みを共有することで、解決の糸口を見つけられる場を提供したいと考え、このテーマを選びました。

### ターゲットユーザ
- ペットを飼っているオーナー
- ペットに興味がある人
- 他のペットオーナーと交流したい人

### 主な利用シーン
- 自分のペットの写真や日常を共有する時
- 他のオーナーのペットの写真を見て癒されたい時
- ペットに関する情報やアドバイスを求めたい時
